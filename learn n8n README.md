# learn-n8n

My daily practice log while learning n8n — going from core concepts to building AI-powered automations, with the goal of freelancing/job-ready skills.

Each folder is one small, focused workflow. Every workflow has its own `README.md` (what it does, concepts covered, how to run it) and a `workflow.json` you can import directly into n8n.

## How to use these workflows

1. Open n8n → click **Add workflow** → click the **three-dot menu (⋯)** → **Import from File**
2. Select the `workflow.json` from any folder below
3. Read that folder's README for context on what it teaches
